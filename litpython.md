Python literario
===========================

Python es un lenguaje excelente para la creación de gráficos y para el cálculo
simbólico, gracias a las librerías NumPy, SciPy y similares.

Probando algo:


~~~~{.python}
def greeting(name):
    greetz = "Hello world #{name}"
    return greetz

greeting("David")
~~~~~~~~~~~~~
